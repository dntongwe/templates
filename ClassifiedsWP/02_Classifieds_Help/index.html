Child Theme

A child theme is commonly used to allow you to update theme to newer versions while preserving any changes that you may have made to its files. In shortest child theme is a theme that inherits functionality and styling of another ( parent ) theme and they are recommended way of modifying any theme.

Because there are already massive amount of tutorials and guides on how to use and set up them we will point you towards few of them and that should be more than enough to make you fully understand their function and the process of using them.

 

  • Official WordPress Guide To Child Themes
  • A Guide To: WordPress Child Themes Development
  • How To Create Child Theme ( Video Tutorial )

Overall Settings

Video Example on how to access theme options

Logo

Here you can setup logo appearance

Site Logo - select your site logo.

Logo Padding - set logo padding if needed or set 0 if it is not. Padding regulates how much space Logo will take from the edge of its container. It takes four numbers with values for TOP RIGHT BOTTOM LEFT sides.

 

Navigation

Here you can setup look of the navigation

Navigation Padding - set navigation padding if needed in the form TOP RIGHT BOTTOM RIGHT ( 0px 0px 0px 0px ) or enter 0 if it is not needed.

Enable Sticky Navigation - enable or disable sticky navigation on the site from the select menu. The sticky navigation, when enabled, means that navigation bar (logo and navigation) will always be visible on the site.

Post Formats

If you have doubts or not so much WordPress experience you can familiarize yourself with post formats by visiting following link.

  • https://codex.wordpress.org/Post_Formats

You can choose your blog post format whenever you are inside given single post where on the right side widget you will have checkboxes to select specific format of the post.

Path would be:

Posts -> All Posts -> "Name of the post" -> Format widget on the right side of screen

 

 

When you choose some post format, below the editor you will notice different fields that are depandable of the chosen format. They are explained in the coming subsections.

Standard

This post format have these options:

Input URL to be embeded - in this field you can input whatever URL you would like to be embed. If this option is empty than featured image will be shown.

Gallery

This post format have these options:

Add images to the gallery - here you can select the images you want to show and also to change their order.

Link

For the link post format you have this option:

Input link - here you can input the link for which you want to be displayed. Additionally you can set its background by setting the featured image.

Image

This post format have no addition options since it will use the featured image that you have selected.

Video

This post format have these options:

Input video URL - here you can input the URL to the self hosted or video which will be embedded.

Select video type - select if the provided URL is from the self hosted video or from the video which should be embedded.

Quote

For the quote post format you have these options:

Input the quotation - here you can type the quotation for which you want to be displayed.

Input the quoted person's name  - input the full name of the person who is being quoted.

You can also set the background image by selecting the featured image.

Status

This post format have no additional options since it will use the featured image that you have selected.

Audio

This post format have these options:

Input URL for the audio - input the URL to the audio file.

Select type of the audio - select if the audio file is self hosted or if it needs to be embedded (supported format for self hosted audio is mp3).

Additionaly to this if you set featured image and you input the auio URL the image will be displayed in the bakground.

Classified Ads WordPress theme is using three custom post types that you can access by following below shown paths:

 

Ads

Ads

This post type is used for your ads. Everything that user puts on the form in the front end you will be able to see it here and edit if needed. Once user submits an ad it must be approved after which you will receive mail about new add with link to the edit page so you can review it and approve or decline it.

Beside basic things like title description and image of the add you also have these fields:

Ad Expire Time - this is auto populated once you publish the add ( it successfully passes the review ) but you can also change it on your own.

Ad Price - here you put price for the ad but without the symbol, just plain number.

Discounted Ad Price - here you put discounted price of the ad if there is any and again just plain number without symbol.

Call For Info - if this is checked then 'Call For Info' is displayed instead of price

Ad Location - on the map you are able to set location of the ad - you can type desired location in the field and also click and drag marker on the map.

Ad Videos - here you put videos for the ad ( in the theme options you can control max number of videos which can be added on the frontend form while here you can put as many as you like )

Ad Images - here you put iamges for the ad ( in the theme options you can control max number of iamge which can be added on the frontend form while here you can put as many as you like )

Ad Display Map? - enable or disable display of the map with location on the ad single.

Ad Is Featured? - make this ad featured or not. Featured ads are always listed first on the listing of the ads.

Ad Phone - if this field is populated then this value will be displayed, otherwise it will use the phone that is set in the user profile.

Ad Views - this field is visible only by you and represents number of clicks on the ad single page ( this is auto incremented ).

Ad Paid - this field will indicate if the user have paid for the ad or not.

Ad Visibility - this field is used to disable the ad in case that submiter decides that it should be hidden for some reason.

For the custom fields and selecting categories for the AD read at the section 'Custom Fields'

Custom Fields

In order to add custom fields go to Custom Fields ->Add New

Input title for the name of the custom fields and select for which categories are you creating custom fields

After this you can start adding custom fields

There are two types of custom fields:

  • Input field
  • Select field
    • Standard select field
    • Conditional select field

Input field

Once you click on the add new field by default input field is being selected and you only need to input name for that field and label for it.

When inputing name of the field do not put spaces or special chars just clean letters and underscore while for the label you can put anything since this is waht users and you see.

Standard Select

In order to create select field for the 'Field Type' select 'Select Field' and populate 'Field Name' and 'Field label' like it is described for the standard input field.

For the values of the select box put VALUE|LABEL one per line and make sure that VALUE does not have empty spaces and special chars but only letters/numbers/underscore

 

Conditional Select Field

We will explain conditional select box on the example of creating custom fields for category Vehicles where you will certainly need car manufacturer and car model and where car model values are dependable of the value of the car manufacturer. So to begin:

Create one standard select for the car manufacturer like on the image bellow:

Now click on the button 'Make Subgroups' and you will notice additional fields created. These fields will help you to specify models for the specific car manufacturer.

Populate 'Subgroup Name' field in the same way as 'Field Name' and 'Subgroup Label' like 'Field Label' and after that populate values for the models for each car manufacturer in the same form as for the standard select box which was already explained.

After this you can click on Publish.

 

Logic of displaying custom fields for ads is the same in the backend and in frontend ,and because of that, here we will focus just on the backend.

Go to Ads->Add New and scroll to the bottom of the page.

You will see the box for the custom fields that will be emty until you select categories which have custom fields.

Note here that displayed fields are from the last category, so if parent category has fields and child don't ( while both being selected for the ad ) no custom fields will be displayed, so make sure to select both when adding the custom fields.

In order to set custom fields here go the the category selection box and select categories for which you have created custom fields and once you scroll back down again you will see custom fields of that category 

Once you finish with populating all the fields you can go ahead and click Publish / Edit button.

 

Important Notice: If you want to delete custom field make sure to delete it from trash also!
